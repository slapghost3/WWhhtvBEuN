# 项目简介

## 核心功能
- 用户系统：注册登录、JWT认证、权限管理、个人中心
- 民宿浏览：首页展示、列表浏览、详情查看、图片展示
- 搜索筛选：关键词搜索、城市筛选、价格筛选、房型筛选
- 收藏功能：添加收藏、取消收藏、收藏列表、收藏统计
- 评论系统：查看评论、评分展示、评论统计、情感分析
- 智能推荐：协同过滤算法、热门推荐、个性化推荐
- 情感分析：SnowNLP情感打分、正面负面分类、词云展示
- 数据可视化：ECharts图表、民宿分布地图、价格分析
- 管理后台：用户管理、民宿管理、评论管理、数据统计
- 爬虫模块：Scrapy框架、反爬策略、数据清洗、自动入库

## 适用场景
- 民宿行业信息展示与推荐
- 旅游规划与预订辅助
- 民宿运营数据分析与管理

## 技术特点
- 前端：Vue 3、Element Plus、ECharts、Vite
- 后端：Django 5.1、Django REST Framework、JWT
- 数据库：MySQL 8.0
- 架构：前后端分离、RESTful API
